This website is currently an **Astro** blog deployed to **Cloudflare Pages**. All editorial content is committed as markdown. Images are hosted on Cloudflare's **R2 Object Storage**.

The website's current design was implemented with plenty of help from **Claude Code** but is an evolution of a fussy theme I build myself years ago. The display font is the _wonky_ variant of **Fraunces**. The body fonts are all the **IBM Plex Sans** and **IBM Plex Mono** families. All fonts are hosted by Google Fonts. Some decorative images are public domain assets discovered via _The Public Domain Review's_ **[Public Domain Image Archive](https://pdimagearchive.org/)**.
